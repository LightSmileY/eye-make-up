# eyeshow一个以眼妆学习和分享为主题的AI、社交、电商相结合的平台

[http://eyeshow.fengblog.xyz/]: 



### 安装依赖

```
yarn install
```

### 开发时运行

```
yarn run serve
```

### 打包构建
```
yarn run build
```

### 运行测试
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

（若未安装yarn，可换为node.js自带的npm。如：npm install）
